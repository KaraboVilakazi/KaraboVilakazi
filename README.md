### 🔑 Engineering Highlights

- ⚡ **Concurrency control** — optimistic locking with retry (SwiftCart) and deadlock-safe pessimistic locking via UUID-ordered acquisition (Imali)
- 🔄 **CI/CD pipelines** — GitHub Actions across all backend projects; deploy pipeline for AWS ECS Fargate
- ☁️ **Infrastructure as Code** — Terraform provisioning for ECS, RDS, ElastiCache, ALB, ECR on AWS
- 🔐 **Production security** — JWT auth, RBAC, environment-based secrets, Spring Security
- 🧪 **Integration testing** — H2-backed test profiles, 19 integration tests covering auth, accounts, and transactions
- 📊 **Audit & observability** — transaction audit logs with `REQUIRES_NEW` propagation so no failure can erase a trail
